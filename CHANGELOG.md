# Changelog

## v0.1.0

- Added the standalone `clawspec` package with `validate`, `run`, `init`, and `coverage` commands.
- Extracted the 19 shipped assertion types for artifact, behavioral, handoff, identity, integration, operational, permission, precondition, semantic, and tool checks.
- Added schema validation, template expansion, scenario discovery, run orchestration, and consistency handling.
- Added the Python API surface: `validate()`, `run()`, `init()`, `coverage()`, `ClawspecConfig`, and the report models.
- Added coverage ledger parsing and JSON summary generation.
- Added the OpenClaw skill wrapper in `skill/`.
- Added runnable public examples in `examples/basic/` and `examples/multi-agent/`.
- Added a redacted Ceremonia showcase in `examples/ceremonia/`.
- Added public documentation for methodology, migration, pipelines, quick start, and the assertion reference.
