# Ceremonia Showcase

This directory is a redacted showcase extracted from `openclaw-ceremonia`.

It is intentionally incomplete and sanitized:

- Secrets, tokens, private URLs, and internal sender addresses have been removed.
- Only representative structures are preserved.
- The goal is to show contract shapes, not to ship a runnable production environment.

Included examples:

- `skills/newsletter/` — skill contract example
- `skills/webinar-orchestrator/` — orchestration and pipeline example
- `skills/campaign-workflow/` — second orchestration-style skill example
- `agents/marketing/brand/` — agent structure example
- `agents/marketing/email/` — second agent structure example
- `coverage-ledger.yaml` — redacted project coverage ledger
- `baselines/marketing-core-wave-1.yaml` — sample baseline report snapshot

Use this folder to study contract structure, naming conventions, and documentation patterns.
